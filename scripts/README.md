# Scripts
Active script categories:
- ops/: operational inspection scripts for jobs and image status
- maintenance/: maintenance scripts that update content or generate missing assets
- `clone_postgres_db.ps1`: clone the current development PostgreSQL database into test/prod targets
- `promote_test_to_prod.ps1`: build and verify test, migrate prod, then roll prod forward

Archived script categories were moved to `archive/scripts/`.
