Beginner What is Document Manager?

The Document Manager serves as your central repository for all writing-related files. Import existing documents, research materials, and reference content to make them accessible to AI features and searchable within your creative workflow.

📄 File Storage

Upload and store documents, PDFs, images, and text files.

🔍 Content Search

Find specific information across all your uploaded documents.

🤖 AI Context

Make documents available as context for AI assistance.

📋 Organization

Categorize and tag documents for easy management.

Multiple ways to add documents to your library:

  • Cloud Integration: Connect Google Drive, Dropbox, or OneDrive

Document Processing Pipeline ▼

What happens when you upload a document:

  • Upload: File transferred securely to storage
  • Content Extraction: Text extracted from various file formats
  • Indexing: Content made searchable using full-text indexing
  • Embedding Generation: Vector embeddings created for AI similarity search
  • Metadata Analysis: File properties and content characteristics analyzed
  • Preview Generation: Thumbnails and previews created for images and documents
⏱️
Processing Time: Large documents may take several minutes to process. You can continue working while processing completes in the background.
