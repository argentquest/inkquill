Beginner What is the World List?

The World List is your dashboard for managing all your creative universes. Each world contains characters, locations, lore items, and stories - everything you need for rich storytelling.

Creating Your First World

2 minutes
  1. 1
    Click the "Create New World" button
  2. 2
    Enter a name and description for your world
  3. 3
    Choose privacy settings (private, shared, or public)
  4. 4
    Click "Create" to build your universe!
💡
Pro Tip: Start with a simple concept for your first world. You can always expand and add complexity later as you become more familiar with the tools.

Understanding World Cards

Each world appears as a card showing:

  • Title & Description: Your world's name and summary
  • Element Counts: How many characters, locations, lore items, and stories you've created
  • Privacy Status: Whether the world is private, shared, or public
  • Last Updated: When you last made changes

Intermediate World Management Features

Quick Actions

Edit, duplicate, or delete worlds directly from the list. Hover over a world card to see action buttons.

Privacy Controls

Toggle between private (only you), shared (specific users), or public (everyone can view).

Search & Filter

Use the search bar to find worlds by name or description. Filter by privacy level or creation date.

World Templates

Start with pre-built templates for common genres like fantasy, sci-fi, or contemporary fiction.

Element Count Indicators

The colored badges on each world card show:

Badge Element Type What It Contains
👥 Characters People & Beings Protagonists, antagonists, NPCs, and creatures
🏰 Locations Places Cities, buildings, rooms, and regions
📜 Lore Items World Knowledge Magic systems, artifacts, history, and customs
📚 Stories Narratives Complete stories set in this world

Advanced Power User Features

Bulk Operations ▼

Select multiple worlds to perform batch operations:

  • Change privacy settings for multiple worlds
  • Export world data in bulk
  • Archive completed projects
Ctrl+Click to select multiple worlds
Shift+Click to select range
World Duplication Strategies ▼

Use world duplication for:

  • Alternate Timelines: Create "what if" scenarios
  • Version Control: Keep backups before major changes
  • Template Creation: Build reusable world foundations
⚠️
Cost Note: Duplicating worlds with many AI-generated elements will consume credits for regenerating embeddings and search indices.
API Integration ▼

Advanced users can integrate with external tools:

GET /api/worlds
POST /api/worlds
PUT /api/worlds/{id}
DELETE /api/worlds/{id}

Performance Optimization

For large world collections:

  • Pagination: Only 20 worlds load at once by default
  • Lazy Loading: World details load when needed
  • Search Indexing: Real-time search across all content
