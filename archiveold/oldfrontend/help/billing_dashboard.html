The AI Storytelling Assistant uses a credit-based system to manage AI services fairly and sustainably. Credits are consumed only when you use AI-powered features - all basic writing, editing, and organization is completely free.

Factors Affecting Credit Costs ▼

Credit usage varies based on several factors:

  • Content Length: Longer generations cost more credits
  • Complexity: More detailed requests require more processing
  • Context Size: Larger worlds provide more context but use more credits
  • Quality Level: Higher quality settings increase credit costs
  • Revision Rounds: Multiple AI passes on the same content
💰 Cost Preview: The system shows estimated credit costs before executing AI operations, so you're never surprised.

Advanced Cost Optimization Strategies

Efficient AI Usage Patterns ▼

Maximize value from your AI credits:

  • Strategic Timing: Use AI for creative blocks, not routine writing
  • Batch Operations: Process multiple related requests together
  • Selective Generation: Request AI help for specific challenges
  • Edit Don't Regenerate: Modify AI output rather than requesting new content
  • Progressive Refinement: Start with cheaper requests, then enhance selectively

Quality vs Cost Balance ▼

Find the right balance for your needs:

  • Draft Phase: Use lower-cost, faster AI for initial content
  • Revision Phase: Invest in higher-quality AI for polishing
  • Experimental Writing: Try cheaper options when exploring ideas
  • Final Projects: Use premium features for publication-ready work
  • Learning Phase: Start with basic features, upgrade as you improve
Common Cost Traps to Avoid ▼

Expensive habits that drain credits unnecessarily:

  • Over-Generation: Requesting AI content you don't actually use
  • Repetitive Analysis: Running the same consistency checks multiple times
  • Perfectionism: Generating multiple versions instead of editing one
  • Lazy Prompting: Vague requests that produce poor results
  • Feature Overlap: Using multiple features for the same problem
⚠️
Avoid Credit Waste: Always preview credit costs before confirming AI operations. Cancel requests that seem too expensive for the expected value.

Advanced Cost Management

Professional strategies for heavy users:

  • Batch Processing: Group similar requests to improve efficiency
  • Template Development: Create reusable prompts for common tasks
  • Session Planning: Plan AI usage before starting writing sessions
  • Credit Budgeting: Allocate credits per project or writing goal
  • ROI Tracking: Measure value gained per credit spent
💰 Pro Tip: Most successful users spend 60-70% of their credits on writing assistance and 30-40% on content development and analysis.
