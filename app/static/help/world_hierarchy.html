Beginner Understanding the Hierarchy

The World Hierarchy provides a bird's-eye view of your entire world structure. It shows how stories, acts, scenes, characters, locations, and lore items connect and organize within your creative universe.

📚 Stories

Top-level containers for your narratives. Each story can have multiple acts and scenes.

🎭 Acts & Scenes

Story structure elements. Acts contain scenes, providing narrative organization.

👥 Characters

People and beings in your world. Can be associated with any story element.

🏰 Locations

Places in your world. Can have hierarchical relationships (city > building > room).

📜 Lore Items

World knowledge like magic systems, artifacts, history, and customs.

Intermediate Managing Element Associations

🔗 Role-Based Association System

Create meaningful connections between elements with specific roles and context. This helps AI understand relationships for better content generation.

Association Types ▼

You can create associations between any elements:

  • Story ↔ Character: Define character roles in stories (protagonist, antagonist, mentor)
  • Story ↔ Location: Specify which places appear in stories and their significance
  • Story ↔ Lore: Connect relevant world knowledge to narratives
  • Act ↔ Elements: Associate elements with specific story acts
  • Scene ↔ Elements: Define what appears in individual scenes
Example Character Roles:
- Protagonist: Main character driving the story
- Antagonist: Opposition or conflict source  
- Mentor: Guide providing wisdom
- Love Interest: Romantic subplot
- Comic Relief: Provides humor and levity
Creating Associations ▼

Adding Element Associations

2 minutes
  1. 1
    Right-click on any story, act, or scene element
  2. 2
    Select "Manage Associations" from the context menu
  3. 3
    Choose the element type you want to associate
  4. 4
    Select specific elements and define their roles
  5. 5
    Add notes explaining the relationship context
  6. 6
    Save to create bidirectional connections
✨
Best Practice: Be specific with roles and notes. Instead of just "character appears in story," use "protagonist seeking revenge for family death."
AI Benefits of Associations ▼

Well-defined associations help AI provide:

  • Context-Aware Writing: AI knows which characters are available in scenes
  • Consistent Characterization: Maintains character roles across scenes
  • Location-Appropriate Content: Suggests actions fitting the current setting
  • Lore Integration: Weaves relevant world knowledge into narratives
💰 Free Feature: Creating associations doesn't cost credits. AI benefits come when generating content.

Beginner Right-Click Context Actions

Right-clicking on any element in the hierarchy reveals a context menu with quick actions relevant to that element type.

Story Context Menu ▼
  • View Details: Open story detail page
  • Edit Story: Modify story information
  • Add Act: Create new act in this story
  • Manage Associations: Connect characters, locations, lore
  • Generate Synopsis: AI-powered story summary
  • Delete Story: Remove story and all its content
Act Context Menu ▼
  • View/Edit Act: Open act editor with AI assistance
  • Add Scene: Create new scene in this act
  • Manage Associations: Connect elements to this act
  • AI Review: Get AI feedback on act structure
  • Move Up/Down: Reorder act within story
  • Duplicate Act: Create copy for alternate versions
Scene Context Menu ▼
  • Edit Scene: Open scene editor with AI writing assistance
  • Manage Associations: Define which characters/locations appear
  • Scene Analysis: AI breakdown of scene elements
  • Split Scene: Divide into multiple scenes
  • Merge with Next: Combine with following scene
  • Set as Favorite: Mark for quick access
World Element Context Menus ▼

Characters, Locations, and Lore Items:

  • View Details: Open detailed element page
  • Quick Edit: Modify basic information inline
  • Generate Image: AI-powered visual creation
  • Expand Description: AI-enhanced details
  • Find Associations: Show where element is used
  • Add to Story: Quick association with existing stories

Batch Operations

Select multiple elements (Ctrl+Click or Shift+Click) to access batch operations:

  • Bulk Associate: Connect multiple elements to a story/act/scene
  • Batch Export: Download selected elements as JSON/CSV
  • Mass Tag: Apply tags to multiple elements at once
  • Group Delete: Remove multiple elements (with confirmation)
⚠️
Caution: Batch delete operations cannot be undone. Always double-check your selection before confirming.

Advanced Power User Features

Hierarchy Customization ▼

Customize the hierarchy view:

  • Filter by Type: Show only specific element types
  • Sort Options: Alphabetical, creation date, or custom order
  • Depth Limits: Control how many levels expand automatically
  • Color Coding: Visual indicators for completion status
View Settings (Gear Icon):
- Show Associations: Toggle connection indicators
- Compact Mode: Reduce spacing for large hierarchies
- Auto Expand: Set default expansion depth
- Color Scheme: Choose visual theme
Drag and Drop Operations ▼

Move elements by dragging:

  • Reorder Scenes: Drag scenes within acts
  • Move Acts: Reorganize story structure
  • Nest Locations: Create parent-child relationships
  • Quick Associate: Drag elements onto stories/acts/scenes
✨
Visual Feedback: Drop zones highlight when dragging compatible elements. Invalid drops are prevented automatically.
Search and Navigation ▼

Advanced search capabilities:

  • Fuzzy Search: Find elements with partial/misspelled names
  • Property Search: Search within descriptions and notes
  • Association Search: Find elements by their connections
  • Recent Items: Quick access to recently modified elements
Search Syntax:
name:John - Find by exact name
type:character - Filter by element type
role:protagonist - Find by association role
tag:important - Search by tags
modified:today - Recently changed items
Export and Import ▼

Data portability options:

  • Selective Export: Choose specific branches of the hierarchy
  • Format Options: JSON, CSV, or custom formats
  • Include Associations: Export with relationship data
  • Merge Import: Combine with existing world data
