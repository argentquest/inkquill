Import World from Literature

Extract characters, locations, and lore from existing books and literature to create rich, detailed worlds.

Import Process:

  1. Upload your book file (PDF, EPUB, TXT, or DOCX)
  2. Select extraction preferences (characters, locations, lore)
  3. AI analyzes the text and identifies world elements
  4. Review and approve extracted elements
  5. Customize and expand the imported world

Supported Features:

  • Character Extraction: Names, roles, and basic descriptions
  • Location Discovery: Places and settings from the text
  • Lore Identification: Cultural elements, artifacts, and customs
  • Relationship Mapping: Connections between characters
  • Smart Filtering: Excludes minor mentions and background elements
Best Results: Works best with fantasy, sci-fi, and fiction with detailed world-building. Classic literature provides excellent source material.
Copyright Notice: Only upload books you own or have permission to use. Respect copyright laws.
