Import Overview

The Book Import feature allows you to bring existing manuscripts, published works, or reference materials into your storytelling workspace. This powerful tool analyzes your content, extracts structural elements, and integrates everything seamlessly with your world-building and AI assistance features.

Key Capabilities

Automatic Structure Detection
Identifies chapters, scenes, and story elements
Character Extraction
Finds and catalogs characters mentioned in your text
Location Mapping
Identifies settings and locations throughout your work
AI Integration
Makes your content searchable for AI assistance

Supported Formats

Import books in various formats to work with your existing content:

Text Files

.txt (Plain text)
.rtf (Rich Text Format)
.md (Markdown)

Documents

.docx (Microsoft Word)
.odt (OpenDocument)
.pages (Apple Pages)

eBooks

.epub (Electronic Publication)
.mobi (Kindle format)
.pdf (Portable Document)

Format Notes

  • PDF Files: Text must be selectable (not image-based)
  • EPUB Files: DRM-free files only
  • File Size: Maximum 50MB per file
  • Encoding: UTF-8 recommended for best results

Upload Process

Follow these steps to import your book successfully:

Select Your File
Choose your book file from your computer. You can drag and drop or use the file browser.
Provide Metadata
Enter title, author, genre, and description. This helps with organization and analysis.
Choose Import Settings
Select analysis depth, structure detection level, and integration preferences.
Start Import
Begin the import process. Large files may take several minutes to process.

Troubleshooting

Common Issues

File Too Large: Compress or split files larger than 50MB
Unsupported Format: Convert to a supported format first
DRM Protection: Only DRM-free files can be imported
Structure Issues: Check chapter formatting consistency

Performance Tips

Large Files: Import during off-peak hours
Multiple Books: Import one book at a time
Analysis Depth: Reduce depth for faster processing
Progress Tracking: Monitor in background tasks panel
